# Documentation Site

The documentation site is the comprehensive knowledge base for the Waylio platform, built with Mintlify. It provides detailed guides, API references, component documentation, and technical resources for developers, healthcare professionals, and system administrators.

<Info>
The documentation site runs on port 3004 in development. Access it at http://localhost:3004 to browse the complete documentation.
</Info>

## Overview

The documentation site serves as the single source of truth for all Waylio-related information, from getting started guides to detailed API references. It's designed to support different user types with role-specific content and progressive disclosure of information.

### Content Guidelines

<Tabs>
<Tab title="Writing Style">
  **Principles**:
  - Clear and concise language
  - Active voice over passive
  - Present tense for current features
  - Second person ("you") for instructions
  - Consistent terminology
  
  **Structure**:
  - Lead with most important information
  - Use progressive disclosure
  - Include complete examples
  - Provide context and rationale
  - End with next steps
</Tab>

<Tab title="Code Examples">
  **Best Practices**:
  - Complete, runnable examples
  - Real-world scenarios
  - Error handling included
  - Multiple language support
  - Copy-paste ready
  
  **Example Format**:
  ```typescript
  // Complete example with imports and context
  import { WaylioClient } from '@waylio/sdk';
  
  async function createAppointment() {
    try {
      const client = new WaylioClient({
        apiKey: process.env.WAYLIO_API_KEY,
      });
      
      const appointment = await client.appointments.create({
        patientId: 'patient_123',
        providerId: 'provider_456',
        scheduledAt: new Date('2024-02-01T10:00:00Z'),
      });
      
      console.log('Success:', appointment.confirmationNumber);
      return appointment;
    } catch (error) {
      console.error('Failed to create appointment:', error);
      throw error;
    }
  }
  ```
</Tab>
