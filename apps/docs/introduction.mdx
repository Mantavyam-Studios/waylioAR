# Welcome to Waylio

Waylio is a production-grade healthcare technology platform built on a modern turborepo architecture. It provides a comprehensive solution for hospitals to digitize and optimize their operations through intelligent systems.

## Platform Overview

Waylio is built as a **monorepo** containing multiple applications and shared packages, designed for scalability and maintainability:

### Frontend Applications
- **Mobile App** (Kotlin/Swift): Patient-facing mobile application
- **Web Dashboard** (Next.js): Staff interface for healthcare professionals
- **Marketing Website** (Next.js): Public-facing website

### Backend Services
- **API Server** (Next.js Serverless): Core business logic and data processing
- **Email Service** (React Email): Transactional email templates and delivery
- **Development Tools**: Storybook, Database Studio, Documentation

### Shared Infrastructure
- **20+ Packages**: Reusable components, utilities, and integrations
- **Type-Safe APIs**: Full TypeScript coverage across the entire stack
- **Modern DevOps**: Automated testing, deployment, and monitoring

## Technology Stack

Waylio leverages modern, production-ready technologies:

### Core Technologies
- **Frontend**: Next.js 14, React 18, TypeScript
- **Backend**: Next.js Serverless Functions, Prisma ORM
- **Database**: PostgreSQL (Neon), Redis for caching
- **Authentication**: Clerk with multi-factor authentication
- **Styling**: Tailwind CSS, shadcn/ui components

### External Integrations
- **Notifications**: Knock for real-time messaging
- **Email**: Resend with React Email templates
- **Payments**: Stripe for secure transactions
- **AR Navigation**: Multiset AI SDK
- **Analytics**: PostHog, Google Analytics
- **Monitoring**: Sentry, Better Stack

### Development Tools
- **Monorepo**: Turborepo for efficient builds
- **Testing**: Vitest, Playwright for E2E testing
- **Code Quality**: Biome for formatting and linting
- **Documentation**: Mintlify for beautiful docs
- **Deployment**: Vercel with automatic previews
